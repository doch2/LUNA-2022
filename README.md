# 주의사항

`assets/img/awards/` 의 하위 이미지 파일의 이름은 다음 형식을 따른다.

    년도-대회이름(영어는 무조건 소문자)
    
    2019-hello new world.jpg

`assets/img/items/`의 하위 이미지 파일의 이름은 다음 형식을 따른다.

```
년도 이름(영어는 무조건 소문자)

2020 ttv.png
```



`_items/`  하위 md 파일은 다음 형식을 따른다.

```
년도-이름.md

2020-TTV.md
```
